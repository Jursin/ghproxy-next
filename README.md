# Github Proxy

基于 Next.js、React、Tailwind CSS 和 TypeScript 搭建的的 Github Proxy 网站。

## 技术栈

- **框架**: Next.js 16 (App Router)
- **UI 库**: React 19
- **样式**: Tailwind CSS 4
- **语言**: TypeScript
- **图标**: Lucide React

## 开发

安装依赖：
```bash
pnpm install
```

启动开发服务器：
```bash
pnpm run dev
```

访问 [http://localhost:3000](http://localhost:3000) 查看网站。

## 构建

```bash
pnpm run build
```
